# 標題（Headers）的用法

使用 \# 可以建立標題，以下是所有標題的樣子。

# 使用一個 \# 可以建立一級標題

## 使用兩個 \# 可以建立二級標題

### 使用三個 \# 可以建立三級標題

#### 使用四個 \# 可以建立四級標題

##### 使用五個 \# 可以建立五級標題

###### 使用六個 \# 可以建立六級標題，這是最多了，看起來和五級標題也沒什麼區別?

NOTE: 在 GitHub 上看，六級標題會變成灰色。

在文字底下加三個 = 也會變成一級標題
===

在文字底下加三個 - 也會變成二級標題
